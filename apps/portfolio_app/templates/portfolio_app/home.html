TransCrypter

M.E.A.N.

With cryptocurrencies gaining significant mainstream/media attention, there has been an influx of applications in the market dealing with the topic. Myself being an avid crypto-enthusiast, I decided to create a handy application where users can track their crypto trades, and save trades to their very own crypto-portfolio, helping them visualize gains and losses, portfolio allocations, market statistics, etc.

Currently, there isn't a user friendly tool in the market where one can enter in a trade between two crypto-currencies at any point in time, and instantly get the fiat value of that trade at that point in time. If you are a crypto-enthusiast or you are new to cryptos, check out 'Transcrypter'. The web-app gathers real time data using an API integration with cryptoCompare and handles calculations on the back-end to show users their portfolio and trade metrics

Pongster

iOS

An immersive arcade ping-pong game that'll have you hooked in seconds. Built with Swift, the app allows users to select various difficulty levels against the computer algorithm, with the enemy in each level getting progressively faster and having more accurate hits. Users also have the ability to play 2-player against one another.

If you're a fan of classic arcade games, feel free to clone the code on Github and provide any feedback you have!

Record Store

Django

For fans of vinyls and record stores, we created an online platform that would give users access to purchasing their favorite records online. Currently the data is hardcorded for demonstration purposes, however the platform is flexible enough to allow us to work with any vendor to easily add their collection of records to the site.

The platform uses Bootstrap for a slick user interface and is seamlessly integrated with the YouTube API, to allow users to instantly sample a 30 seconds snippet of the records they are buying. Integration with the secure Stripe API allows for fluid credit card processing in a matter of seconds.
